Simple Binary Encoding (SBE)

SBE is an OSI layer 6 presentation for encoding and decoding messages in binary format for low-latency applications. The SBE project on GitHub is the reference implementation for the FIX SBE RC2 standard for encoding of financial messages.

SBE provides a compiler for taking a message schema and generating stubs for messaging parsing in multiple languages. Currently C++, C#, and Java are supported. SBE allows allows for the on-the-fly decoding of messages from a compiler generated meta description. Details for using SBE can be found on the Wiki.

Performance is a key requirement for SBE. The performance is focused on the exchange of messages via a local area network with servers running x86 processors. The design of SBE follows a number of principles and it makes no compromise in adding features that violate these principles. You might find that alternative codecs, such as Protocol Buffers, offer more features such as allowing strings at any location in a message. SBE makes some restrictions on such use for the return of ~25X increased throughput in message encoding and decoding.
